CHEMCONNECT design philosphy is in line with FAIR data practices which provides guidelines for data definition and management for the efficient sharing of scientific data. FAIR is an acronym which stands for:
  • Findable: CHEMCONNECT views as a network of interconnections Ontology provides context and metadata for search. Metadata and data should be easy to find for both humans and computers. Machine-readable metadata are essential for automatic discovery of datasets and services.
  • Accessible: A cloud-based tool for the management of public, consortium and private data. Once the user finds the required data, she/he/they need to know how they can be accessed, possibly including authentication and authorisation.
  • Interoperable: The ontology provides meta-data standardized context and for interconversion. The data usually need to be integrated with other data. In addition, the data need to interoperate with applications or workflows for analysis, storage, and processing.
  • Re-usable: Ontology based on domain standards, both W3C and chemical domain knowledge. The ultimate goal of FAIR is to optimise the reuse of data. To achieve this, metadata and data should be well-described so that they can be replicated and/or combined in different settings.

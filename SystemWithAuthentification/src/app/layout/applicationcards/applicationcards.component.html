Calculations Calculations from 2-D Lewis Structures

Thermodynamics calculations are made using the thermodynamic database of Benson rules (or HBI rules), symmetry structures (optical, internal and external), structural,for example steric or ring structure influenced, and radical corrections (disassociation energies, symmetry losses, etc.).

 Upload Data Upload data to the database

The uploading of data occurs in several steps. First, the raw file is uploaded and stored in the database storage. This file, is then partitioned. The partitioning depends on the type of data that is stored. Each of these partitions are then interpreted and are added to a set corresponding to this data type.

 Dataset Collection Manage the Collections of Datasets

The dataset collection specifies which set of fundamental data should be used for the thermodyanmic calculations. A single dataset collection specifies which combination of fundamental data should be used for the calculations
